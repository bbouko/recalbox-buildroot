## RECALBOX - SYSTEM MAME ##

Placez ici vos roms mame.

Les roms doivent avoir l'extension ".zip"

Les roms doivent provenir du set 0.37b

Le fichier clrmamepro/imame4all_gamelist.txt liste les jeux compatibles.

Vous pouvez utiliser clrmamepro disponible sur http://mamedev.emulab.it/clrmamepro/
et utiliser le fichier .dat dans le repertoire clrmamepro pour verifier la version de vos roms.

Vous pouvez télécharger les roms mame en recherchant sur google :
https://www.google.fr/webhp?ie=UTF-8#safe=off&q=mame4all%200.37b5%20complete%20romset%202270
