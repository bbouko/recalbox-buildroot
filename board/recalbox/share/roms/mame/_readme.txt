## RECALBOX - SYSTEM MAME ##

Put your mame roms in this directory.

Rom files must have a ".zip" extension.

Compatible roms must come from set 0.37b5

The file clrmamepro/imame4all_gamelist.txt list compatible games.

You can use clrmamepro available at http://mamedev.emulab.it/clrmamepro/ and use the .dat file in clrmamepro directory to check your roms.

You can download mame roms by searching on google :
https://www.google.fr/webhp?ie=UTF-8#safe=off&q=mame4all%200.37b5%20complete%20romset%202270
